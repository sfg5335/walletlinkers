Step 1: Install Node.js (2 minutes)
────────────────────────────────────
The download page should have opened in your browser.
If not, go to: https://nodejs.org/en/download/

1. Click "Download Node.js (LTS)"
2. Run the downloaded .pkg file
3. Follow the installer (just click Next/Continue)
4. It will install automatically


Step 2: Verify Installation (10 seconds)
─────────────────────────────────────────
Open a NEW terminal window and run:

node --version

You should see version numbers like:
v20.x.x

Step 3: Deploy Your App (3 minutes)

cd /Users/phillipyon/Downloads/CollectEVM-master
./final-deploy.sh
vercel
